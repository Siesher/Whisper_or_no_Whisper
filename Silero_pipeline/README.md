# End-to-End Silero Fine-Tuning Project

Этот проект представляет собой удобный "оркестратор" для дообучения (fine-tuning) моделей Silero для задачи автоматического распознавания речи (ASR).

## Ключевая идея

Процесс дообучения Silero требует работы с их официальным репозиторием и специфическим форматом данных. Этот проект автоматизирует все рутинные шаги:

1.  **Авто-настройка:** Автоматически клонирует репозиторий Silero и скачивает базовую модель.
2.  **Авто-подготовка данных:** Конвертирует ваш простой JSON-файл в формат `train.txt`/`test.txt`, который требует Silero.
3.  **Авто-конфигурация:** Генерирует сложный YAML-конфиг для обучения на основе ваших простых настроек в `config.yaml`.
4.  **Авто-запуск:** Запускает процесс обучения Silero одной командой.
