# End-to-End Silero Fine-Tuning Project

Этот проект представляет собой удобный "оркестратор" для дообучения (fine-tuning) моделей Silero для задачи автоматического распознавания речи (ASR).

## Ключевая идея

Процесс дообучения Silero требует работы с их официальным репозиторием и специфическим форматом данных. Этот проект автоматизирует все рутинные шаги:

1.  **Авто-настройка:** Автоматически клонирует репозиторий Silero и скачивает базовую модель.
2.  **Авто-подготовка данных:** Конвертирует ваш простой JSON-файл в формат `train.txt`/`test.txt`, который требует Silero.
3.  **Авто-конфигурация:** Генерирует сложный YAML-конфиг для обучения на основе ваших простых настроек в `config.yaml`.
4.  **Авто-запуск:** Запускает процесс обучения Silero одной командой.

## Установка и запуск

**1. Установка зависимостей:**

Убедитесь, что у вас установлен `git`. Затем установите Python-библиотеки:

```bash
pip install -r requirements.txt

**2. Подготовка данных:**
-   Поместите ваши аудиофайлы (.wav, моно, 16кГц) в папку audio_files/.
-   Заполните файл my_dataset.json, указав пути к аудио и их транскрипции.

**3. Настройка конфигурации:**

Откройте файл config.yaml и настройте его. Вы можете выбрать URL базовой модели и задать основные параметры обучения.

**4. Запуск пайплайна:**

Запустите основной скрипт из терминала:

python run_pipeline.py --config config.yaml
Что произойдет после запуска:
-   Будет создана папка silero-models.
-   Внутри нее будет создана папка prepared_data с файлами train.txt, test.txt, chars.txt.
-   Будет сгенерирован файл generated_finetune_config.yaml.
-   Запустится процесс обучения train.py из репозитория Silero. Вы увидите логи обучения в консоли.
-   Результаты (чекпоинты и логи) будут сохранены в папке silero-models/work_dir_finetune.
