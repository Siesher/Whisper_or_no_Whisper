Инструкции по запуску (`README.md`)

1.  **Настройка окружения:**
    *   Создайте виртуальное окружение: `python -m venv venv && source venv/bin/activate`
    *   Установите зависимости: `pip install -r requirements.txt`
    *   Залогиньтесь в W&B: `wandb login`
    *   Залогиньтесь в Hugging Face: `huggingface-cli login`

2.  **Конфигурация:**
    *   Откройте `config.yaml` и **обязательно** впишите ваш `YOUR_WANDB_USERNAME`.
    *   Проверьте все пути, особенно `audio_directory`.

3.  **Запуск:**
    *   Положите ваши аудиофайлы и `clear_rez_transcriptions.csv` в нужные места.
    *   Выполните главный скрипт:
        
bash
        python main.py --config config.yaml
        `

### Что будет происходить:

Скрипт последовательно выполнит все этапы, указанные в `pipeline_control`. Он будет создавать чекпоинт-файлы в папке `.checkpoints`, поэтому при повторном запуске (например, после сбоя) он пропустит уже завершенные шаги. Все важные результаты (данные, модели, метрики) будут версионироваться и отображаться в вашем дашборде Weights & Biases, предоставляя вам полный контроль и воспроизводимость.
